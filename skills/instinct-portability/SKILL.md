# Instinct Portability

## Purpose
Export learned instincts to a shareable file and import others' — continuous learning across people, not just sessions.

## When to use
When sharing judgement with a teammate or the community.

## How it fits PQA
The human side of continuous learning; turns the instinct library into a shared, growing asset (and a team-tier feature).
It serves the loop (frame -> superpose -> collide -> collapse -> precipitate) and the governing principle: spread probability mass into divergent
and low-probability branches, then collapse it onto whatever survives attack and verification.
